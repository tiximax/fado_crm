👥 Nhập Dữ Liệu Khách Hàng

📁
Chọn file Excel để nhập khách hàng
Hỗ trợ: .xlsx, .xls (Tối đa 10MB)

Yêu cầu cột trong Excel: Họ tên, Email, Số điện thoai (bắt buộc). Địa chỉ (tùy chọn).

🛍️ Nhập Dữ Liệu Sản Phẩm

📦
Chọn file Excel để nhập sản phẩm
Hỗ trợ: .xlsx, .xls (Tối đa 10MB)

Yêu cầu cột trong Excel: Tên sản phẩm, Giá bán (bắt buộc). Mô tả, Danh mục, Trọng lượng, Quốc gia nguồn (tùy chọn).
